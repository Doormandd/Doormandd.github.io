- 用途：工作流程图示

## 注意事项

这些图片目前是占位符，需要根据实际演示界面创建截图。可以使用浏览器截图工具获取实际界面图片。

## 创建方法

1. 打开演示页面：https://Doormandd.github.io/demo/ara-flight-planner/
2. 执行完整的飞行路径规划流程
3. 在关键步骤截图：
   - 绘制ROI多边形
   - 生成边界矩形
   - 生成飞行路径
   - 路径裁剪结果
4. 使用图片编辑工具调整尺寸并添加必要的标注

## 图片优化建议

- 使用WebP格式以提高加载速度
- 压缩图片以减少文件大小
- 确保图片在不同设备上的显示效果
- 添加alt文本以提高可访问性
